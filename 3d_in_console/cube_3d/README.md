# Console 3D Cube Rotation

A Python-based 3D visualization engine that renders a rotating wireframe cube directly in the terminal using rotation matrices and perspective projection.

## Overview
This project demonstrates the application of linear algebra in computer graphics. It calculates 3D vertex coordinates, applies transformation matrices for rotation, and projects the resulting points onto a 2D coordinate system for console output.

## Key Features
- Matrix Transformations: Implementation of rotation matrices for X, Y, and Z axes.
- Perspective Projection: Conversion of 3D space coordinates into 2D screen space.
- Pure Python: Developed using only standard libraries (math, time, os).
- Object-Oriented Design: Clear separation between the mathematical model and the rendering logic.
