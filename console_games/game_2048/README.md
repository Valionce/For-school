# 2048: The Nine Challenge

A lightweight, console-based puzzle game inspired by the classic 2048, but with a faster pace and a unique progression system.

## The Concept
Instead of the traditional powers of two (2, 4, 8...), this version uses a linear scale from **1 to 9**. 
- Merge two identical numbers to get the next one (e.g., `1 + 1 = 2`, `2 + 2 = 3`).
- Your ultimate goal is to reach the number **9** to win the game.

## Features
- **Pure Python:** No external dependencies required.
- **Terminal UI:** Play directly in your console.
- **Classic Mechanics:** Use WASD or Arrow keys to slide tiles.
