Console Chess
A lightweight and efficient CLI chess implementation written in Python. Play a quick match against a friend directly in your terminal.
Features
Pure Python: No external dependencies required.
Modern Build: Developed in 2025.
Core Mechanics: All pieces and their standard movement patterns are fully implemented.
Known Limitations (Roadmap)
Currently, the project focuses on board mechanics and move validation. The following logic is not yet implemented:
Automatic Check detection.
Checkmate and Stalemate conditions.
